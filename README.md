# Decisive

## User Story
AS A person who has a hard time making decisions
I WANT an app that can give me weather and location readings
SO THAT I can determine whether to go out or not.

-------------------
## Built With
* HTML
* JavaScript
* Tailwind CSS framework
* Openweather API
* Mapbox API
